# mini Blockchain

I am building a mini blockchain from scratch to understand how it works at a fundamental level. This is for educational purposes.

---

## What is a Blockchain?

Imagine you and your friends are keeping track of who owes who money. Instead of one person holding the notebook (and potentially cheating), everyone has a copy of the same notebook. Every time someone writes a new entry, everyone checks it and adds it to their own copy.

That's basically a blockchain. It's a list of records (called blocks) where:

- Each record knows about the one before it
- If anyone tries to change an old record, everyone else can tell something is wrong
- No single person controls it

The "chain" part is because every block is linked to the one before it, like a literal chain.
